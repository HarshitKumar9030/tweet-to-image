# Tweet to Image
Tweet to Image — Download a Tweet as an Instagram-Ready Image

# How to use 
- Go to the website - https://www.tweet-to-image.tech/
- Fill the textboxes
- Then Download! 
- Then you can upload it to platforms like Instagram, Facebook etc
